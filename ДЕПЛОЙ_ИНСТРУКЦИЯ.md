## 📱 Получение APK:

### Вариант 1: GitHub Actions (автоматически)
1. Перейти: https://github.com/neiroset-7373/qwerty2/actions
2. Выбрать последний запуск "Build Android APK"
3. Скачать артефакт `app-debug`
4. Распаковать ZIP → получить `app-debug.apk`

### Вариант 2: GitHub Releases (по тегу)
```bash
git tag v1.1.3
git push origin v1.1.3
```
APK появится в: https://github.com/neiroset-7373/qwerty2/releases
